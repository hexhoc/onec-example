# Конвертация данных 2

Это конфигурация которая описывает:
- Правила конвертации объектов и свойств
- Правила выгрузки данных
- Правила регистрации объектов

Весь этот набор правил используется для организации обмена **только между системами на платформе 1С**

Гибкость механизмов позволяет сделать почти любой обмен данными используя произвольные запросы и выгрузку

## Простое использование (без плана обмена и БСП)

### Вручную

1. Открываем обработку "Универсальный обмен данными в формате XML (Обработки)". Она в основном либо встроена в конфигурацию, либо находится в пакете с поставкой "Конвертация данных 2"
2. Подгружаем правила обмена
3. Выбираем нужные данные, период выгрузки и выгружаем их
4. На стороне приемника так же открываем эту обработку, и на вкладке "Загрузка данных" загружаем ранее выгруженные данные.
