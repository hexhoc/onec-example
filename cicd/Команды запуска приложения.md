
# Команды запуска приложения

## Обновление расширения из хранилища и выгрузка снимка в файл

```cmd
"C:\Program Files\1cv8\8.3.27.2074\bin\1cv8.exe DESIGNER "/S SERVER-1C:4541/base_name" 
/UC update 
/ConfigurationRepositoryBindCfg 
/ConfigurationRepositoryF "tcp://1cgit/base_name_extension" 
/ConfigurationRepositoryN "Админ" 
/ConfigurationRepositoryP "123456" 
/ConfigurationRepositoryDumpCfg "\\1cgit\Release\base_name\ext\base_name_extension.cfe" ConfigurationRepositoryUpdateCfg -force -Extension "base_name_extension" /DisableStartupDialogs"
```

## Выгрузка расширения в XML

```powershell
Start-Process -FilePath "C:\Program Files\1cv8\8.3.27.2074\bin\1cv8.exe" -ArgumentList 'DESIGNER "/S SERVER-1C:4541/base_name" /UC update /DumpConfigToFiles "C:\GIT\onec\my_project" -Extension "ИмяРасширения" -Force /DisableStartupDialogs /Out "C:\GIT\onec\out.txt"' -Wait -NoNewWindow
```

## Конвертация XML в CFE

```powershell
Start-Process -FilePath "C:\Program Files\1cv8\8.3.27.2074\bin\1cv8.exe" -ArgumentList 'DESIGNER "/S SERVER-1C:4541/base_name" /UC update /LoadConfigFromFiles "C:\GIT\onec\my_project" -Extension "ИмяРасширения" /DumpCfg "C:\GIT\onec\ИмяРасширения.cfe" -Extension "РасширениеЗУПКорп" /DisableStartupDialogs /Out "C:\GIT\onec\zup\out.txt"' -Wait -NoNewWindow
```
